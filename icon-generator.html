🎨 마세라티 아이콘 생성 도구

180x180 (iPad)
192x192 (Android)
512x512 (고해상도)

📋 사용 방법

  1. SVG 파일 선택 버튼을 클릭하여 trident_mobile_white.svg 파일을 선택합니다.
  2. 자동으로 3가지 크기의 PNG 아이콘이 생성됩니다.
  3. 각 아이콘의 다운로드 버튼을 클릭하여 저장합니다:
    • icon-180.png (iPad 홈화면용)
    • icon-192.png (Android용)
    • icon-512.png (고해상도용)
  4. 다운로드한 PNG 파일들을 img/ 폴더에 넣습니다.
  5. GitHub에 모든 파일을 업로드합니다.

📱 아이패드에서 홈화면 추가

  1. Safari에서 퀴즈 페이지를 엽니다.
  2. 공유 버튼(위쪽 화살표)을 탭합니다.
  3. "홈 화면에 추가"를 선택합니다.
  4. 이름을 확인하고 "추가"를 탭합니다.
  5. 홈화면에 마세라티 로고 아이콘이 나타납니다! 🎉
